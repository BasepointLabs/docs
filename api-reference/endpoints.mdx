This page summarizes the main customer-facing API routes. Admin-only operator routes are intentionally excluded.

## Health and waitlist

| Method | Path | Purpose |
| --- | --- | --- |
| `GET` | `/health` | Check service health |
| `POST` | `/waitlist` | Join the marketing waitlist |

## AI employees and chat

| Method | Path | Purpose |
| --- | --- | --- |
| `GET` | `/agent/tools` | List native tools |
| `POST` | `/agent/chat` | Legacy or default agent chat endpoint |
| `GET` | `/agents` | List AI employees |
| `POST` | `/agents` | Create an AI employee |
| `GET` | `/agents/:id` | Get one AI employee |
| `PATCH` | `/agents/:id` | Update an AI employee |
| `DELETE` | `/agents/:id` | Delete an AI employee |
| `PUT` | `/agents/:id/skills` | Set employee skills |
| `POST` | `/agents/:id/chat` | Chat with an employee |
| `GET` | `/agents/:id/messages` | List messages |
| `DELETE` | `/agents/:id/messages` | Clear messages |
| `GET` | `/agent-templates` | List AI employee templates |
| `GET` | `/llm/models` | List supported models |

## Knowledge and skills

| Method | Path | Purpose |
| --- | --- | --- |
| `POST` | `/knowledge` | Create a knowledge base |

| `POST` | `/knowledge/:id/documents/url` | Add URL document |

| `POST` | `/knowledge/:id/documents/:docId/retry` | Retry ingestion |
| `POST` | `/knowledge/:id/documents/:docId/recrawl` | Recrawl URL document |
| `POST` | `/knowledge/:id/documents/:docId/move` | Move document |
| `DELETE` | `/knowledge/:id/documents/:docId` | Delete document |
| `GET` | `/skills` | List skills |
| `POST` | `/skills` | Create skill |
| `GET` | `/skills/:id` | Get skill |
| `PATCH` | `/skills/:id` | Update skill |
| `DELETE` | `/skills/:id` | Delete skill |
